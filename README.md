# Laboratorio 4: Cliente Móvil con Consumo de API RESTful 

Este proyecto es una aplicación móvil desarrollada en **React Native** con **Expo**, diseñada para demostrar el consumo de servicios web mediante una arquitectura cliente-servidor.

La aplicación gestiona un catálogo de biblioteca (Autores, Géneros y Libros) realizando peticiones HTTP asíncronas para las operaciones CRUD.

## Funcionalidades Principales

* **Navegación:** Flujo de pantallas mediante React Navigation.
* **Consumo de Datos:** Uso de `fetch` para operaciones GET, POST, PUT y DELETE.
* **Gestión de Estado:** Implementación de Hooks (`useState`, `useEffect`, `useFocusEffect`) para la reactividad de la interfaz.
* **Interfaz de Usuario:** Listados dinámicos, formularios de ingreso y alertas de confirmación nativas.

## Requisitos Previos

* Node.js instalado.
* Aplicación **Expo Go** instalada en tu celular o un emulador de Android/iOS configurado.
